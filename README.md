# AI-Powered-Campus-Placement-Management-System

# Users

Students - The Future Employees 
Placement Co-ordinator - Admins
Super Admin - HOD

# Student Module 

Basic Provisional Flow :
  Student Profile Management - Register, Login, Forgot Password, Reset Password, Home Page It's CRUD Operation.
  Student Will Upload the Resume --> Resume Analysis --> Skill Assessment --> Final Result/Output -->Auto Suggestion of Companies ->Interview Schedule --> Placement Status Tracking

  Additional Features : 
    Interview Preparation Suggestion along with Practice Question Generation.(Difficulty Level Wise).

   --> can also add the additional certificates which will enhance the profile of the Students. 

# AI Module Process which Totally Going to be run in the Backend

It will ask for the Job Description from Admin or any Placement - Co ordinator

Resume will Match with the Job Description  --> ATS Score Card will be Generated along with the Skill Gap Analysis and Suggesgation of More Effective or Better Resume --> Result Score generation.

# Placement Co-ordinator Module

  Create The Placement Drive --> Manage Students --> Generate Reports and Excel along with the mail functionlity to directly email the Admin (HOD).
  
  Auto Verification process of KT from KT Records which will be manually entered by the students. (it will help to fill important Details with Accuracy in the User Marksheet Data).
  
  Having the functionlity to apply different kind of Filters Placement Commitee Members can manage Students as well!


# Admin -  HOD

  Manage the Placement Co-ordinators and Communicate with the Company.
  Status Update to Co-ordinator coming from the Company.
